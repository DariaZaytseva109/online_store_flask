API для интернет-магазина на FLASK

реализованы:
	a. апи для создания продукта;
	b. апи для получения страницы продуктов;
	c. апи для получения продукта по его ID;


— запустить сервер:  python main.py
— создать продукт: 
	URL: "/api/v1/product/"
	направить POST-запрос, в теле JSON вида: {"id": "4", "name": "Notebook", "price": 200}
— получить все продукты:
	URL: "/api/v1/product/"
	направить GET-запрос, параметры вида: ?limit=5&page=1
— получить продукт по id:
	URL: "/api/v1/product/<id>/, например /api/v1/product/5/
	направить GET-запрос

тестировала через Postman
